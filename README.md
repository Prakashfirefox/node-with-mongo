# node-with-mongo

node js with mongo db get post put delete

## Installation

Install mongo migrate

```bash

  npm i

```

## need local mongodb

or mention your db url in .env file

```bash

MONGO_URI=mongodb://localhost:27017/org-app

```

## Run Migration

```bash

 npm run migrate

```

## Run Seeder

```bash

 npm run seed

```

## Start server

```bash

 npm start

```
